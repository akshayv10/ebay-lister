# API listing report

For preparation, lead with confirmation that exactly two complete offers exist but remain unpublished. Link both AliExpress sources and report title, category ID, deterministic SKU/offer IDs, variation prices, EPS image count, fee response, policies, location key, General 10% readiness, and the exact run ID. End by saying a new explicit approval is required to publish.

For successful publication, report both canonical eBay URLs, listing IDs, General campaign/ad IDs, verified 10.0% bids, Priority off, and history updates.

For failure, lead with `publish_rolled_back` or `reconciliation_required`; list which ads were deleted, which listings/groups were withdrawn, and every uncertain outcome. Never claim live state unless both listings and promotions were independently verified.
